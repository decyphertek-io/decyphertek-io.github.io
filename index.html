Security Features

Our virtual machines are built with security in mind, offering a range of features to protect your infrastructure.

Debian / Rocky based VMs

  • Nginx Reverse Proxy
  • Apache Reverse Proxy
  • Modsecurity WAF
  • UFW Host Firewall
  • Crowdsec IPS
  • OSSEC HIDS
  • Rsyslog Collection
  • Automated Updates
  • Auditd Monitoring

Flatcar Linux based VMs

  • Immutable OS
  • Auto-updating OS
  • Containerized Server
  • Nginx Reverse Proxy
  • Manage via Portainer
